## Executing Single-File

```bash
java HelloJava.java
```

### Passing Arguments

```bash
java PassingArguments.java World!
```

### Multiple Classes

```bash
java MultipleClasses.java
```

### ReferenceJDK

```bash
java ReferenceJDK.java
```

### ReferenceNonJDK

```bash
java -cp ./commons-lang3-3.17.0.jar ReferenceNonJDK.java
```
